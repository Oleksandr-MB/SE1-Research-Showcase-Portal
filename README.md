# SE1-Research-Showcase-Portal

### Purpose: 
Researchers create profiles and showcase their research projects, including abstracts, papers, and demo tools.

### CI
- From repo root:
  - Backend tests: `.venv/bin/python -m unittest discover -s tst -p 'test_*.py'`
  - Frontend typecheck: `npm --prefix src/frontend run typecheck`
- If you `cd src/frontend` first:
  - Frontend typecheck: `npm run typecheck`
- Details: `docs/CI_CD.md`
